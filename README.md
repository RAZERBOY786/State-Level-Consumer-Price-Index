# State-Level Consumer Price Index (CPI) Analytics Engine

An immersive, widescreen macroeconomic cockpit designed to analyze and visualize structural inflation data across states and union territories in India. Built with a high-end **Glassmorphism UI layout design concept**, this dashboard transforms massive government datasets into high-contrast, interactive geospatial insights.

---

## 📊 Project Visual Dimensions

The analytics architecture is mapped across 5 core investigative modules to provide an immediate macro-to-micro structural breakdown:

*   **📦 KPI Cards → Summary:** High-performance calculation wrappers outputting real-time National Baseline Averages, automatic Year-over-Year (YoY) delta percentages, and peak workload structural anomalies.
*   **📈 Line Chart → Trend:** Multi-series temporal linear mapping visualizing multi-year inflation vector directions across major corridors.
*   **📊 Bar Chart → Ranking:** Ascending cross-sectional rank scaling comparing state indexes relative to baseline averages.
*   **🗺️ Map → State Visualization:** A dynamic, data-responsive interactive choropleth map tracking localized pricing density and regional corridors without boundary clipping.
*   **🔥 Heatmap → Pattern:** A cluster configuration density matrix tracking parameter variations across temporal milestones (Core, Food, Energy, Housing vectors).

---

## 🌐 Official Dataset Context

The analytical dataset powering this platform is derived directly from the Government of India's open portal registry:

*   **Data Source Link:** [State-Level Consumer Price Index (Rural/Urban) up to February 2024](https://www.data.gov.in/resource/state-level-consumer-price-index-ruralurban-upto-february-2024)
*   **Publisher:** Ministry of Statistics and Programme Implementation (MoSPI), Central Statistics Office (CSO).
*   **Coverage Dimensions:** Monthly granular price indexes across Rural, Urban, and Combined sectors from 2011 through early 2024.
*   **Methodological Note:** Early 2011 records contain higher indices for high-altitude zones (e.g., Sikkim at 178.4). This reflects the historical baseline configuration adjustments deployed by the CSO before scaling everything into the standardized Base 2012 = 100 timeline matrix framework.

## 🛠️ Core Technology Stack

*   **Framework Core:** React 19 + TypeScript (TSX)
*   **Build Assembly Wrapper:** Vite
*   **Styling Engine:** Tailwind CSS + Custom Blurring Backlighting Filters
*   **CSV Engine Parser:** PapaParse
*   **Geospatial Layer Matrix:** `@vishalvoid/react-india-map` (Bounded Centroid Layout)
